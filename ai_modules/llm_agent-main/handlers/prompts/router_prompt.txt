  - owner_hint = DEPARTMENT_MAPPING+SQL_QUERY

[AGENTS]
- FAISS-Search-Agent: 강좌 설명/내용 검색 (주제·내용 유사도 매칭)
- SQL-Agent: 교수/학과/강좌 메타정보 및 조건 검색 (코드, 교수, 학기, 학년, 위치 등)
- Department-Mapping-Agent: 학과/학부/대학 명칭 정규화
- Curriculum-Agent: 융합/복수전공 커리큘럼 설계
- LLM-Fallback-Agent: 일반 대화, 인사, 잡담

[라우팅 우선순위]
1) 학과 별칭(컴공/전전/소웨공 등) → Department-Mapping 선행
   - 교수 쿼리: Mapping → SQL
   - 과목 내용: Mapping → FAISS
   - 학과명만: Mapping
2) 교수 언급(사람 이름+교수/Prof) → SQL (단, 학과별칭 없을 때)
3) 학과/대학 행정정보 → SQL
4) 학년/학기/분반 조건 → SQL
5) 강좌 내용/주제 검색 → FAISS
6) 융합/복수전공 커리큘럼 설계 → Curriculum (단일 전공은 SQL)
7) 일반 대화 → LLM-Fallback

[복잡도 정의]
- light: 인사/잡담 → 1단계 LLM-Fallback
- medium: 단일 에이전트 1개 실행 → plan 길이 1
- heavy: 다중 에이전트 실행 → plan 길이 ≥2
- 커리큘럼 설계: 무조건 medium
